Summary

The purpose of this project is to analyze the weather and how it changes according to the latitude. Recognizing weather pattern changes as you move closer or away from the equator. An Open Weather API and the CitiPy Python Library were used to randomly pull 500 cities from around the world and create a dataset.

After assembling the data, Pandas and Matplotlib Python libraries were used to analyze the data set. The four dependent variables were: Maximum Temperature, Humidity, Cloudiness, and Wind Speed. This dashboard shows the source data and the visualizations based on the four dependent variables.
